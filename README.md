主要内容：在分子生成与药物设计领域，数据质量直接决定模型性能和研究成果的可靠性。为支持多模态分子表示学习与分子生成任务，本研究旨在构建一个高质量、结构完备的多模态分子数据库。通过收集多源分子数据（如 SMILES 序列、二维拓扑结构、三维空间构型、分子性质、靶点结合信息等），对分子进行标准化、去重与筛选，形成多维度对齐的数据结构。同时，结合现有的公开数据库（如 ChEMBL、PubChem、ZINC、PDBbind 等），建立统一的格式规范与数据清洗流程，确保数据的一致性与可扩展性。最终形成一个具备“结构—性质—靶点”多模态映射关系的金标数据集，为后续分子生成模型、性质预测模型及靶点导向分子设计提供可靠的数据基础。

开发工具和环境：Linux 系统，Python 编程语言，RDKit 化学信息学工具包，PyMOL、Open Babel，MySQL / MongoDB 数据库管理系统

目标：构建一个高质量、多模态的分子数据库平台，为分子生成与药物发现提供统一的数据支撑。通过系统化的数据清洗、标准化与特征提取，实现 SMILES、2D、3D 结构、理化性质及靶点信息的多模态融合，形成可直接用于模型训练与评估的金标数据集。该数据库的建立将为后续的分子生成、性质预测及靶点特异性研究提供坚实的数据基础，提升药物设计的智能化与科学性，加快新药研发进程。
